Sobre o CropDox

O Sistema CropDox

Objetivo

CropDox é uma Aplicação Web que oferece Software como Serviço (SaaS) destinada a auxiliar em diversas rotinas de escritório relacionadas a digitalização de documentos. Dentre os objetivos do projeto, o principal, está relacionado ao desenvolvimento de funcionalidades inovadoras para o processamento e organização de imagens.

Funcionalidade Inicial

Nesta versão inicial, o sistema possui uma funcionalidade bastante prática que consiste em transmitir fotos tiradas do App diretamente para o navegador através da leitura de um código QR. Para isto, basta instalar o aplicativo Android disponível na Play Store e seguir as instruções dele.
