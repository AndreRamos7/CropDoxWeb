Sobre o CropDox

O Sistema CropDox

Objetivo

CropDox é uma Aplicação Web projetado para oferecer Software como Serviço (SaaS) destinada a auxiliar em diversas rotinas relacionadas a digitalização de documentos. Dentre os objetivos do projeto, o principal, está relacionado ao desenvolvimento de funcionalidades inovadoras para o processamento e organização de dados e imagens.

Funcionalidade Inicial

Por enquanto, nesta versão inicial, o sistema possui uma funcionalidade bastante prática que consiste em transmitir fotos tiradas do App diretamente para o navegador através da leitura de um código QR. Para isto, basta instalar o aplicativo Android disponível na Play Store e seguir as instruções dele.
