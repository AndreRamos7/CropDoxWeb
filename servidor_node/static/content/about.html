Sobre o CropDox

O Sistema CropDox
TTTT

CropDox é uma aplicação Web que oferece Software como Serviço (SaaS) destinado a auxiliar em diversas rotinas de escritório. Dentre os objetivos do projeto, o principal está relacionado ao desenvolvimento de funcionalidade para o processamento de imagens e aplicações inovadoras que utilizem Inteligência Artificial e Visualização da Informação.

Funcionalidades iniciais

Nesta versão inicial, o sistema possui uma funcionalidade bastante prática que consiste em transmitir fotos tiradas do App diretamente para o navegador através da leitura de um código QR.
