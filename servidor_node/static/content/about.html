Sobre o CropDox

O Sistema CropDox

Objetivo

CropDox é uma Aplicação Web que oferece Software como Serviço (SaaS) destinada a auxiliar em diversas rotinas de escritório. Dentre os objetivos do projeto, o principal, está relacionado ao desenvolvimento de funcionalidades para o processamento de imagens e aplicações inovadoras.

Funcionalidade Inicial

Nesta versão inicial, o sistema possui uma funcionalidade bastante prática que consiste em transmitir fotos tiradas do App diretamente para o navegador através da leitura de um código QR. Para isto, basta instalar o aplicativo Android disponível na Play Store e seguir as instruções dele. Também conta com ferramentas extras como remover acentos de textos e calcular datas.
